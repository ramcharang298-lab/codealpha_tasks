# Telco Customer Churn Analysis Dashboard

## Project Overview
This project is an interactive Power BI dashboard developed to analyze customer churn in a telecommunications company.

## Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel

## Dashboard Features
- Total Customers KPI
- Customers Churned KPI
- Churn Rate
- Average Tenure
- Average Monthly Charges
- Contract Type Analysis
- Payment Method Analysis
- Internet Service Analysis
- Gender-wise Analysis
- Senior Citizen Analysis

## Skills Demonstrated
- Data Visualization
- Dashboard Design
- KPI Creation
- Data Analysis
