# Task 2 - Exploratory Data Analysis (EDA)

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Telco Customer Churn dataset using Microsoft Excel. The objective is to understand customer behavior, identify churn patterns, and generate business insights through pivot tables and pivot charts.

## Dataset

Telco Customer Churn Dataset

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Exploratory Data Analysis (EDA)

## Analysis Performed

- Data Cleaning & Profiling
- Customer Churn by Gender
- Customer Churn by Contract Type
- Customer Churn by Payment Method
- Customer Churn by Internet Service
- Customer Tenure Analysis
- Average Monthly Charges by Churn Status

## Key Insights

- Month-to-Month contract customers have the highest churn.
- Electronic Check users show a higher churn rate.
- Fiber Optic customers have a comparatively higher churn rate.
- Customers with longer tenure are more likely to stay.
- Customers with higher monthly charges tend to churn more.
